前言

requests是一个流行的Python第三方库，用于简化HTTP请求的发送和处理。它提供了一组简洁而强大的API，使得发送HTTP请求和处理响应变得简单和直观。

功能与特点

  1. 简洁而直观的API：requests库的API设计简单而直观，易于使用和理解。它提供了一组类似于HTTP方法（如GET、POST、PUT、DELETE）的函数，使得发送不同类型的请求变得简单明了。
  2. 支持常见的HTTP操作：requests库支持发送各种类型的HTTP请求，包括GET、POST、PUT、DELETE等。您可以轻松地指定URL、请求头、查询参数、请求体和其他相关信息。
  3. 自动化的会话管理：requests库提供了会话（Session）对象，使得在多个请求之间共享会话状态和Cookie变得容易。这对于模拟登录、保持会话状态和处理带有身份验证的请求非常有用。
  4. 内置的身份验证支持：requests库支持各种身份验证机制，包括基本身份验证（Basic Authentication）、摘要身份验证（Digest Authentication）和OAuth等。您可以轻松地提供身份验证凭据以进行请求。
  5. 强大的响应处理功能：requests库提供了丰富的功能来处理HTTP响应。您可以访问响应的状态码、头部信息和内容。它还提供了方便的方法来处理JSON响应、文件下载和流式响应等。
  6. 自动化的重定向处理：requests库自动处理HTTP重定向，并提供了配置选项来控制重定向的行为。您可以选择允许或禁止重定向，并设置最大重定向次数。
  7. 文件上传和下载支持：requests库支持文件上传和下载操作。您可以轻松地发送文件作为请求体或下载文件到本地文件系统。
  8. 可定制的请求和会话配置：requests库提供了一系列配置选项，使您能够自定义请求和会话的行为。您可以设置超时时间、代理、SSL验证、连接池大小等。

总之，requests库的特点在于其简单易用的API设计、丰富的功能和灵活性。它成为了Python开发人员发送和处理HTTP请求的首选工具之一，并在Web开发、数据爬取和API交互等场景中广泛应用。

requests函数库的使用

1.发送GET请求:

import requests
# 使用requests发送一个get请求
response = requests.get(url)
# 打印相应信心
print(response.text)

2.发送POST请求：

import requests
# 请求参数
data = {'username': 'user', 'password': 'pass'}
# 发送post请求
response = requests.post(url, data=data)
# 打印了响应的状态码
print(response.status_code)

其他用法

import requests

response = requests.get("https://www.baidu.com")

# 打印响应对象的类型
print(type(response))
# 输出：<class 'requests.models.Response'>

# 打印响应的状态码
print(response.status_code)
# 输出：200

# 打印响应内容的类型
print(type(response.text))
# 输出：<class 'str'>

# 打印响应头内容
print(response.headers)

# 打印响应头中的Content-Type字段值
print(response.headers['content-type'])

# 打印请求头内容
print(response.request.headers)

# 将响应对象的编码设置为utf-8
response.encoding = "utf-8"

# 打印响应内容（文本形式）
print(response.text)

# 打印响应的Cookies
print(response.cookies)

# 打印响应内容（字节形式）
print(response.content)

# 将字节形式的响应内容解码为utf-8编码的字符串
print(response.content.decode("utf-8"))

批量下载图片实例：


import os.path
import response
from lxml import etree

# 定义了目标网页的URL和请求头信息
index_url = "https://sc.chinaz.com/tupian/fengjing_6.html"
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 '
                  '(KHTML, like Gecko) Chrome/113.0.0.0 Safari/537.36'
}

# requests.get()函数发送GET请求并获取响应
response = requests.get(index_url, headers)
# 将响应内容的编码设置为UTF-8
response.encoding = "utf-8"
# 使用response.text获取响应内容的文本形式，并将其赋值给变量text
text = response.text
# print(text)

# etree.HTML()函数创建一个XPath解析对象selector，以便后续的XPath查询和数据提取
selector = etree.HTML(text)

# 使用XPath表达式 //div/img/@data-original 和 //div/img/@alt 分别从网页中提取图片链接和图片的alt属性值
image_urls = selector.xpath('//div/img/@data-original')
image_alts = selector.xpath('//div/img/@alt')

# 定义一个文件写入函数 'wb' 是打开文件的模式参数，其中 'w' 表示写入模式，而 'b' 表示二进制模式
def write_to_file(filename, content):
    with open(filename, 'wb') as file:
        file.write(content)
        
# 保存哪个目录下        
directory = "C:\\Users\\37656\\Desktop\\新建文件夹\\image\\img\\pachong"

for url,alt in  zip(image_urls,image_alts):
    print("https:"+url)
    image_content = requests.get("https:"+url).content
    file_path = os.path.join(directory,'{}.jpg'.format(alt))
    write_to_file(file_path,image_content)

下载虎扑球员信息实例

import requests
from lxml import etree

# 定义了目标网页的URL和请求头信息
index_url = "https://nba.hupu.com/stats/players"
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 '
                  '(KHTML, like Gecko) Chrome/113.0.0.0 Safari/537.36'
}

response = requests.get(index_url, headers=headers)

selectors = etree.HTML(response.text)

tags = selectors.xpath('//table/tbody/tr[@class="color_font1 bg_a"]/td')

ids = selectors.xpath('//table/tbody/tr[position() >= 2]/td[1]')
names = selectors.xpath('//table/tbody/tr[position() >= 2]/td[2]/a')
teams = selectors.xpath('//table/tbody/tr[position() >= 2]/td[3]/a')
scores = selectors.xpath('//table/tbody/tr[position() >= 2]/td[4]')
hit_shoots = selectors.xpath('//table/tbody/tr[position() >= 2]/td[5]')
hit_rates = selectors.xpath('//table/tbody/tr[position() >= 2]/td[6]')
hit_three_points = selectors.xpath('//table/tbody/tr[position() >= 2]/td[7]')
three_point_hit_rates = selectors.xpath('//table/tbody/tr[position() >= 2]/td[8]')
hits_free_throws = selectors.xpath('//table/tbody/tr[position() >= 2]/td[9]')
free_throw_percentages = selectors.xpath('//table/tbody/tr[position() >= 2]/td[10]')
sessions = selectors.xpath('//table/tbody/tr[position() >= 2]/td[11]')
minutes = selectors.xpath('//table/tbody/tr[position() >= 2]/td[12]')
for tag in tags:
    print(tag.text, end=' ')
print()
for id, name, team, score, hit_shoot, hit_rate, hit_three_point, three_point_hit_rate, hits_free_throw, free_throw_percentage, session, minute in \
        zip(ids, names, teams, scores, hit_shoots, hit_rates, hit_three_points, three_point_hit_rates, hits_free_throws,
            free_throw_percentages, sessions, minutes):
    print(id.text, name.text, team.text, score.text, hit_shoot.text, hit_rate.text, hit_three_point.text,
          three_point_hit_rate.text,
          hits_free_throw.text, free_throw_percentage.text, session.text, minute.text)
